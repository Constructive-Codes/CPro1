# CPro1 code and results files

This is the repository for CPro1, a system for solving combinatorial design problems by using LLMs to generate large numbers of C programs.

To experiment with CPro1, clone the repository and follow the documentation in CPro1/README.md.

CPro1 was introduced in the following papers, and this repository has the code and the result files for these papers:<br>

Rosin, C.D. (Dec. 2025) "LLM-Generated Search Heuristics Can Solve Open Instances of Combinatorial Design Problems." [MATH-AI Workshop at NeurIPS 2025](https://openreview.net/forum?id=WlXSZiqcbH)<br>
<br>
Rosin, C.D. (May 2025) "Using Reasoning Models to Generate Search Heuristics that Solve Open Instances of Combinatorial Design Problems." [https://arxiv.org/pdf/2505.23881](https://arxiv.org/pdf/2505.23881)<br>
<br>
Rosin, C.D. (January 2025) "Using Code Generation to Solve Open Instances of Combinatorial Design Problems." [https://arxiv.org/abs/2501.17725](https://arxiv.org/abs/2501.17725)<br>

If you find CPro1 useful in your work, please cite at least one of the above papers.<br>

If you need the exact code used for the above papers, check the github Tags in this repo.
<br>

**designs/** has the solutions to open instances from the papers above, along with the code that generated them.

**CPro1/** has the code and documentation for running CPro1.
